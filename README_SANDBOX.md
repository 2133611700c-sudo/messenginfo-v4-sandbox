# Messenginfo V4 — Sandbox

Этот репозиторий — ПЕСОЧНИЦА для новой платформы Messenginfo V4.

- Не содержит боевого кода.
- Не имеет прямого доступа к прод-средам и секретам.
- Используется для разработки архитектуры, протоколов, CI/CD и агентов.

**Правила:**

1. Все изменения только через pull request в `main`.
2. Следовать документу `AGENTS_SANDBOX.md`.
3. Не хранить в репозитории реальные токены, пароли, ключи и конфиденциальные данные.
4. Любая интеграция с боевой системой — только через отдельный Integrator/Messenginfo и PR в прод-репо.

Этот репозиторий можно безопасно использовать для автономной работы агентов.
